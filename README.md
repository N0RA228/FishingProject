# FishingProject
----------------

**Описание**:  Игра - рыбалка.

Игра в которой нужно рисковать: выловить следующую рыбку и возможно проиграть или уйти, оставаив накопленные монеты.

  - **Технологический стек**: Java, Maven.
  - **Статус**:  Release 1.0.
